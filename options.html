Zendesk View Count

This Chrome extension is using the Zendesk API in order to display in the extension's icon the count of tickets from a Zendesk View of your choice. It also notifies you at the defined frequency for any change in the count number. In order to have this working, please ensure that you are logged in on your Zendesk account and that you setup the following values.

Options

Notify about change in your Zendesk view count each minute(s).


The domain of your Zendesk eq domain.zendesk.com

Zendesk ViewID

Check for tickets change each seconds.
